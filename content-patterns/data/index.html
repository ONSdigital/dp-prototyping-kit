Writing about data in bulletins

Writing your introduction

A brief introduction paragraph should explain where the data comes from.

For example

Estimates of employment, unemployment, economic inactivity, hours worked and redundancies are derived from the Labour Force Survey (LFS), a survey of households.

Adding links

This section includes a link to all data related to this release. To help users find useful or related data, you can add links to up to five datasets on this topic.

Each link should include the title of the dataset, followed by a 30-word description of what the dataset contains. The description should start with the most relevant data to the topic covered, mention any other data which could be useful, and identify the source (unless that's clear from the title).

For example

Crime in England and Wales: Annual supplementary tables
Dataset | Released on 19 July 2018
Includes data on crime involving weapons, perceptions of the police, and anti-social behaviour. From the Crime Survey for England and Wales (CSEW).

If you link to the same dataset in a few different places, keep the description as consistent as you can.

Linking to data from other places

You can include links to specific data sources from external sources. Make this clear by including the name of the source in brackets.

For example

A millennium of macroeconomic data (Bank of England)
Data | Released in 2016
Worksheets A49 and A50 include historic estimates of unemployment over hundreds of years.
